Before you write and run Python code, you need to make sure the Python interpreter is on your computer. In this book, you'll start out with Release 3 of Python, the very latest (and best) version of the language.
A release of Python might already be on your computer. Mac OS X comes with Python 2 preinstalled, as do most versions of Linux. Windows, in contrast does not include any release of Python. Let's check your computer for Python 3. Open up a command-line prompt and, if you are Mac OS X or Linux, type:
python3 -V
